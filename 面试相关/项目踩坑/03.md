> antd 描述内容过长时将超出部分隐藏并显示省略号，鼠标放上去可查看全部信息

```js
columns = [
  {
    title: '描述', dataIndex: 'desc', key: "desc",
    onCell: () => {
      return {
        style: {
          maxWidth: 150,
          overflow: 'hidden',
          whiteSpace: 'nowrap',
          textOverflow: 'ellipsis',
          cursor: 'pointer'
        }
      }
    },
    render: (text) => <Tooltip placement="topLeft" title={text}>{text}</Tooltip>
  },
]
```
