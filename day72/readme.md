## typescript 安装 编译
> 安装：`npm install typescript` 或 `yarn add typescript`

> 编译：`tsc xxx.ts`

> 配置：`tsc --init`

## typescript 数据类型
`typescript`中为了使编码更规范，更好维护，增加了类型校验

JS 中数据类型
- 数值类型 (Number)
- 字符类型 (String)
- 布尔类型 (Boolean)
- Null类型 (Null)
- Undefined类型 (Number)
- 对象类型 (Object)

TS 中数据类型
TS 保留JS数据类型
新增了
  - 元组 (Tuple)
  - 枚举 (Enum)
  - Any (Any)
  - Void (Void)
  - Never (Never)
